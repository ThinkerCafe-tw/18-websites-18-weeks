# AI 時代個人網站開發實戰 - 華視教材開發專案

> **華視空中進修學院 115 學年度上學期「網頁設計」課程教材**

## 📋 專案資訊

- **課程名稱**：AI 時代個人網站開發實戰
- **主編/主講**：鄒慶士教授（國立臺北商業大學 資訊與決策科學研究所）
- **協同主講**：湯明軒（Cruz Tang，ThinkerCafe 創辦人）
- **播出單位**：華視教育頻道
- **播出時間**：2026 年 2-6 月
- **課程時長**：18 講次，每講 30 分鐘

---

## 🎯 課程目標

讓空中進修學院學生在 18 週內，學會用 AI 協作工具快速開發出專業級個人/商業網站

---

## 📚 課程架構

### 第一階段：展示型網站基礎（第 1-3 講）
- 第 1 講：✅ 咖啡廳網站 - AI 協作開發入門
- 第 2 講：✅ 健身房網站 - 風格設計與靈感應用
- 第 3 講：✅ 攝影師作品集 - 版本控制與部署實務

### 第二階段：功能型網站進階（第 4-6 講）
- 第 4 講：✅ 診所預約網站 - 線上預約系統入門
- 第 5 講：✅ 美容院會員網站（上）- 會員系統基礎建設
- 第 6 講：✅ 美容院會員網站（下）- 登入、點數與優惠系統

### 第三階段：專業形象與信任建立（第 7 講）
- 第 7 講：✅ 法律事務所網站 - 專業形象與信任感建立

### 第四階段：更多行業實戰（第 8-15 講）
8 個不同行業的網站開發需求與解決方案

### 第五階段：進階整合（第 16-18 講）
複雜功能整合與商業應用策略

## 📖 華視教材格式要求

每一講必須包含：
1. **單元目標**（3-4 個）
2. **主文**（階層式結構：一、（一）、1.、(1)）
3. **重要名詞**（5-8 個）
4. **自我評量或問題與討論**（5 題）
5. **參考文獻/書目**

**字數要求：5,000-7,000 字/講**

## 🚀 如何使用 Claude Code 開發

### 新增 Claude Code 時

1. 打開 Claude Code
2. 選擇這個專案資料夾：`/Users/thinkercafe/Documents/18-websites-18-weeks`
3. **Claude 會自動讀取 `Claude.md` 獲得完整開發指引**

### 開始開發新的一講

告訴 Claude：

```
請幫我開發第 X 講：[主題名稱]
```

Claude 會：
1. 閱讀課程設計原稿
2. 參考已完成講次的範例
3. 按照華視格式產出教材
4. 檢查字數是否符合要求
