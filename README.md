# AI 時代個人網站開發實戰 - 18 個商用網站實戰教材

> 由 Cruz Tang (@tangcruzz) 開發，華視空中進修學院「網頁設計」課程指定教材

## 🎯 這是什麼？

這個專案包含 **18 個真實商業場景的網站開發教材**，每個案例都展示如何用 AI 協作工具在短時間內開發出專業級商用網站。

**適合對象：**
- 想快速建立個人/商業網站的創業者
- 想學習 AI 協作開發的開發者  
- 想接案賺錢的自由工作者
- 對 AI 時代網頁開發感興趣的學習者

---

## 📚 課程架構 (18 講)

### 🎨 第一階段：展示型網站基礎 (第 1-3 講)

學習如何快速建立具有專業視覺效果的展示型網站

- ✅ 第 1 講：咖啡廳網站 - AI 協作開發入門
- ✅ 第 2 講：健身房網站 - 風格設計與靈感應用
- ✅ 第 3 講：攝影師作品集 - 版本控制與部署實務

### ⚙️ 第二階段：功能型網站進階 (第 4-6 講)

掌握預約系統、會員系統等常見商用功能

- ✅ 第 4 講：診所預約網站 - 線上預約系統入門
- ✅ 第 5 講：美容院會員網站 (上) - 會員系統基礎建設
- ✅ 第 6 講：美容院會員網站 (下) - 登入、點數與優惠系統

### 🏢 第三階段：專業服務業網站 (第 7 講)

學習建立專業形象與信任感

- ✅ 第 7 講：法律事務所網站 - 專業形象與信任感建立

### 💼 第四階段：更多行業實戰 (第 8-15 講)

涵蓋 8 個不同產業的網站開發需求

- ✅ 第 8 講：餐廳訂位網站 - 即時座位與訂位管理
- 📝 第 9 講：瑜伽教室網站 - 課程管理系統
- 📝 第 10 講：寵物美容網站 - 服務展示與預約
- 📝 第 11 講：個人教練網站 - 個人品牌建立
- 📝 第 12 講：民宿訂房網站 - 訂房系統開發
- 📝 第 13 講：家教媒合網站 - 媒合平台開發
- 📝 第 14 講：汽車保養網站 - 客戶管理系統
- 📝 第 15 講：才藝教室網站 - 招生報名系統

### 🚀 第五階段：進階整合 (第 16-18 講)

複雜功能整合與商業應用策略

- 📝 第 16 講：諮商預約網站 - 專業諮商網站
- 📝 第 17 講：活動報名網站 - 活動管理系統
- 📝 第 18 講：顧問預約網站 - 金流整合實務

## ✨ 課程特色

### 🆚 與傳統課程的差異

| 比較項目 | 傳統網頁課程 | 本課程 |
|---------|-------------|--------|
| 學習重點 | HTML/CSS/JavaScript 語法 | 商業需求分析與 AI 協作溝通 |
| 開發方式 | 手寫程式碼，逐行除錯 | 自然語言描述，AI 自動生成 |

| 產出成果 | 教學用示範網站 | 18 個可立即商用的真實案例 |

### 🛠️ 技術棧

本教材使用的技術：
- **前端框架**：React + Tailwind CSS
- **雲端資料庫**：Supabase
- **部署平台**：Vercel
- **版本控制**：Git + GitHub
- **AI 協作工具**：Claude Code

## 🚀 快速開始

### 1. Clone 這個專案

```bash
git clone https://github.com/ThinkerCafe-tw/18-websites-18-weeks.git
cd 18-websites-18-weeks
```

### 2. 開始學習

教材位於 `01-教材內容/` 資料夾，建議依序學習：

```
01-教材內容/
├── 第01講-咖啡廳網站.md
├── 第02講-健身房網站.md
├── 第03講-攝影師作品集.md
└── ...
```

### 3. 實作練習

每一講都包含完整的開發流程，跟著步驟操作即可建立出相同的網站。
