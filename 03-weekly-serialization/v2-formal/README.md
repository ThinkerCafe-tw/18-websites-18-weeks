# 函授週刊 v2（空院正式規格版）

依《115 學年度上學期主編老師工作說明》第 2 節〈函授週刊編寫〉規格製作。

## 規格遵循

| 項目 | 規格要求 | 本版遵循 |
|---|---|---|
| 期數 | 每學期共 6 期 | 6 期 |
| 每期結構 | 課業輔導 + 習題解答 | 二段結構 |
| 課業輔導字數 | ≤1500 字（純中文） | 每期控制於 1,400–1,500 字 |
| 習題數 | 約 10 題 | 每期 10 題 |
| 習題解答 | 完整內容，不得寫「參考教材第 oo 頁」 | 每題完整解析 |
| 課業輔導內容原則 | 摘要重點、補充教材不足、實例分析、法條時事更新 | 三段式：主題定位／重點整理／實例分析 |
| 內容來源 | 避免直接轉貼教材內文 | 所有本體為原創補充 |

## 6 期與教材 18 講對應

| 期次 | 對應講次 | 對應行業 | 主題 |
|---|---|---|---|
| 第一期 | 第 1–3 講 | 咖啡廳 / 健身房 / 攝影工作室 | HTML/CSS 基礎、品牌視覺與作品集結構 |
| 第二期 | 第 4–6 講 | 家醫診所 / 美容院（上）（下） | 預約系統、會員註冊與個人資料保護 |
| 第三期 | 第 7–9 講 | 法律事務所 / 餐廳 / 瑜伽教室 | 專業信任、訂位衝突處理與排程系統 |
| 第四期 | 第 10–12 講 | 寵物美容 / 個人教練 / 民宿 | 個人品牌、自有渠道與收益管理 |
| 第五期 | 第 13–15 講 | 家教媒合 / 汽車保養 / 才藝教室 | 雙邊市場、客戶回流與招生流程 |
| 第六期 | 第 16–18 講 | 諮商 / 活動報名 / 顧問 | 隱私設計、自動化通知與金流整合 |

## 與 v1（老街版）的關係

- v1 老街版 18 期現存於 `../W01-W18-*.md`，因期數與結構不符空院規格，不作為正式交付物
- v1 之情境敘事若合適且字數允許，僅以「實例分析」段落之形式選擇性納入 v2，但不得作為主幹
- v1 檔案保留於 `hanshou-weekly/` 根目錄，不予刪除
