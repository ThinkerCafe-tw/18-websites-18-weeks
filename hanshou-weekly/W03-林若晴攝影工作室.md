**主講老師**：湯明軒 / 鄒慶士
**配合講次**：第三講 — 攝影師作品集：從電腦走向全世界
**發行日**：2026 年 2 月

### 這週我走進第三家店

上週離開健身房之後，我沿著街往北走，走到一個巷口，看到一扇舊木框玻璃門，門上貼著一張小牛皮紙，用鉛筆寫：「若晴攝影工作室，入內請輕聲」。我站了兩秒才推門進去。

裡面很安靜，先聞到的不是咖啡味，是一種淡淡的藥水味——後來才知道是暗房的定影液。整面北窗，光從高處斜斜地落下來，照在一張木頭長桌上。桌上攤著幾張剛洗出來的照片，還沒乾透，邊緣微微捲起。

牆上的照片沒有裱框，就用小夾子夾在一條細繩上，像晾衣服一樣晾著。有一張是老先生坐在自家門口削竹子，眼睛沒看鏡頭；有一張是清晨的市場，魚販正在往攤子上潑水。都不是那種「一看就是作品」的照片，它們更像是某個人願意為你停下來的幾秒鐘。

老闆是一位叫林若晴的女士，52 歲。她泡了茶給我，坐下來說：「你是第七個這個月走進來的陌生人，你們都很輕。」

### GitHub 是什麼？其實是存在的證明

這週主教材教你 GitHub 和 Vercel——把網站從電腦搬上網路。書裡用「倉庫」和「房東」來比喻。那個比喻是對的，但我想再往下走一層。

你這輩子做過的東西大多是「暫時的」。寫的 Word 檔在 D 槽某個資料夾、拍的照片在手機裡、做的 PPT 交出去之後就不知道在誰的郵箱。它們都活著，但沒有一個地方。你不能指著某個東西說「這是我的，它永遠在那裡」。

GitHub 給你的不是一個倉庫，是**一個地址**。一個永遠在那裡的地址。

我這麼跟林若晴形容：以前你要讓別人看作品，要帶著底片本跑去給對方看，對方看完還給你。作品永遠跟著你走，沒有自己的家。有了網站不一樣——作品住在一個地方，全世界任何一個人，半夜三點想起「那個在市場拍魚販的攝影師叫什麼」，輸入網址，門就開了。你不用在，作品自己在招呼他。

林若晴聽到這裡眼睛亮了一下。她說她這輩子搬過七次家，每次最捨不得的就是舊照片。「我總覺得照片跟著我搬來搬去，它們會累。」我那一刻聽懂了——照片想要一個家。

所以主教材那五個步驟（註冊、建倉庫、傳檔案、連 Vercel、按 Deploy）不是技術流程，是**落戶程序**。你在幫你這輩子做的東西登記戶口。

你的網址就是你的門牌。掛上去的那一秒，這個世界上多了一家店。這家店不會因為電腦壞掉而消失，不會因為 IG 改演算法而沉底。只要每年續一次網域費用（一年三四百塊），它就一直在那裡。

### 卡在這裡很正常

我猜你翻開第三講主教材的時候，心裡最大的卡點不是「GitHub 怎麼註冊」，是另一個更安靜的問題——**「可是我沒有東西可以放啊。」**

你想：我又不是攝影師，拍的照片就是家人、小孩、出去玩的風景。我又不是設計師、不是插畫家、不是什麼有「作品」的人。那我做這個網站要放什麼？

我想告訴你一件事：**林若晴也覺得自己沒東西可放**。

她剛離婚那年 47 歲，兩個小孩都大了，發現自己不知道要做什麼。她拿起相機是因為女兒一句「媽你去拍點東西嘛」。剛開始每次看完自己拍的都想刪掉——「別人拍得比我好太多了」「我連構圖是什麼都不知道」。她拍了整整一年都沒給任何人看。

後來怎麼走出來的？她說：「有一天我坐在市場邊等朋友，看到魚販潑水，光很漂亮，我拍了。回家看那張照片的時候我哭了一下——不是因為拍得好，是因為**我發現我真的停下來看了那十秒鐘**。我這輩子很少真的停下來看東西。那張照片是我停下來的證明。」

你的「作品」不一定要多厲害。它可以是：你讀過的書單、你記得的食譜、你寫給小孩的信、你退休後養的盆栽每週長多高、你媽媽傳給你的菜怎麼做。這些都是「你停下來過」的證明，都值得有一個門牌號碼。

技術是一樣的——任何一個想留下一點東西的人，都可以用這個方法蓋一家店。你只是還沒想清楚店裡放什麼。先學會蓋，再慢慢想。店蓋在那裡，東西會慢慢長出來。

### 一扇窗

這週我想推薦你一本書，Susan Sontag 的《論攝影》（On Photography）。1977 年出版，不薄也不好讀，但你不用從頭讀到尾。翻到任何一頁讀兩三段，都會有收穫。

Sontag 寫這本書不是教你怎麼拍照。她在問一個更根本的問題：**我們為什麼要拍？**

我跟林若晴講這本書，她說她讀過，讀完之後「不敢拍照三個月」。Sontag 把她心裡一個很不安的東西講出來了：**我有資格留下這個時刻嗎？**

後來她繼續拍了。她說她想通一件事——**不是我有資格留下，是這個時刻值得有人停下來，而我剛好在**。

你這週學做網站的時候，心裡可能也會有同樣的不安：我有資格開這家店嗎？我有什麼好展示的？答案跟林若晴一樣——不是你有資格，是你這輩子活過的這些日子值得有一個門牌號碼，而你剛好有空學到了這個技術。

如果書太難，可以改看 Wim Wenders 的電影《完美的日子》（Perfect Days，2023）。那是一個東京清潔工每天用舊底片相機拍樹葉的故事，幾乎沒有對白，但你會明白：把時刻留下來，不需要理由。

### 下週見 / 街的另一頭

下週我會走進第四家店，一間小小的中醫診所。那裡有一位女醫師，她的診所裡放著一台老電視，永遠開著沒聲音的新聞台。我想去問她為什麼。

這週你不用急著學完整個 GitHub 和 Vercel 流程。你只要做一件事——想一想，你心裡那個「還沒有門牌號碼的東西」是什麼？可能是一段文字、一組照片、一個食譜、一份清單，都沒關係。想到就好，不用寫下來。

另外，街的另一頭那盞燈還留著。thinker.cafe 那裡這週有人開始分享他們的「第一個網址」——大部分都是很簡單的一頁，但每一個都有故事。你不去也沒關係，像林若晴跟我說的那句話——「只要它在那裡，我知道有人在看世界的其他角落就好」。

那麼，下週見。

——湯明軒 / 2026 春
