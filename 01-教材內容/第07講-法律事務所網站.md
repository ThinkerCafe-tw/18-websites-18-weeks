### 7-1 專業服務業的網站需求:為什麼「信任」大於一切

**一、回到展示型網站,但層次更高**

```

經過第 4-6 講的高強度功能開發後:

(二)法律事務所網站的特殊性

**法律服務 ≠ 咖啡廳**

```
咖啡廳網站的目的:
- 吸引客人來店
- 展示美味的食物
- 營造溫馨氛圍
→ 重點:視覺吸引力

法律事務所網站的目的:
- 建立專業形象
- 證明解決問題的能力
- 讓客戶「敢」委託
→ 重點:信任感
```

**為什麼信任感這麼重要?**

假設你遇到法律問題:
- 被房東無故扣押金
- 車禍理賠糾紛
- 離婚財產分配

你會找誰?

**選項 A:網站看起來很花俏,但沒有案例說明**
→ 你會想:「這律師有經驗嗎?能處理我的問題嗎?」

**選項 B:網站看起來很專業,有清楚的專長領域和成功案例**
→ 你會想:「這律師擅長這類案件,應該可以信任」

**差別就在「信任感」!**

**二、專業服務業的共同特徵**

(一)什麼是「專業服務業」?

不只是律師,還包括:
- 會計師事務所
- 建築師事務所
- 醫療診所(特別是自費項目)
- 管理顧問公司
- 心理諮商所
- 財務顧問公司

**共同特徵**:
1. **高度專業性**:需要專業證照或長期訓練
2. **無形服務**:賣的是知識、經驗、判斷
3. **高風險決策**:客戶的選擇可能影響重大
4. **資訊不對稱**:客戶不懂專業,只能「相信」

**因此網站的任務是**:
- 降低資訊不對稱(讓客戶理解你的專業)
- 建立信任(證明你有能力解決問題)
- 降低決策風險(展示成功案例)

---

(二)專業服務業網站的核心元素

| 元素 | 咖啡廳網站 | 法律事務所網站 |
|------|-----------|---------------|
| 視覺風格 | 溫馨、輕鬆 | 專業、穩重 |
| 配色選擇 | 暖色系(棕、橘) | 冷色系(藍、灰) |
| 核心內容 | 菜單、環境照片 | 專業資歷、案例 |
| 行動呼籲 | 「立即訂位」 | 「免費諮詢」 |
| 信任建立 | Google 評論 | 證照、判決書 |
| 文字風格 | 簡短、口語 | 專業、但不艱澀 |

**關鍵差異**:
- 咖啡廳:讓人「想去」
- 法律事務所:讓人「敢去」

### 7-2 建立信任感的設計元素

**三、視覺設計:專業感的第一印象**

(一)配色的心理學

**專業服務業推薦配色**:

```
【深藍色系】- 最推薦
- 主色:#1e3a8a (深藍)
- 輔助色:#3b82f6 (亮藍)
- 背景:#f8fafc (淺灰)

心理效果:
✓ 信任感(銀行、保險都用藍色)
✓ 專業感(不浮誇、穩重)
✓ 安全感(讓人覺得可靠)

【深灰色系】- 次選
- 主色:#1f2937 (深灰)
- 輔助色:#6b7280 (中灰)
- 點綴:#f59e0b (金色)

心理效果:
✓ 高級感(像精品)
✓ 沉穩感(不會太冷淡)
✓ 金色點綴增加「價值感」

❌ 避免的配色:
- 鮮紅色(太激進)
- 粉紅色(不夠專業)
- 螢光色(不嚴肅)
```

**為什麼藍色最適合?**

研究顯示:
- 藍色是「最受信任」的顏色
- 銀行、保險、醫療都大量使用藍色
- 藍色讓人聯想到「可靠」、「穩定」

### 7-3 案例展示:用故事建立信任

**五、成功案例的黃金公式**

(一)為什麼案例這麼重要?

**數據會說話**:

根據尼爾森調查(Nielsen Study):
- 92% 的消費者相信「其他人的經驗」
- 88% 的消費者會看案例/評論才做決定
- 有案例展示的網站,轉換率提升 3-5 倍

**對法律服務來說更重要**:
- 法律是高風險決策(選錯律師可能輸官司)
- 客戶無法「試用」(不能先打半場官司看看)
- 只能透過「過去的表現」判斷能力

**所以,案例展示是最重要的信任建立工具!**

---

(二)案例展示的結構:STAR 法則

**STAR = Situation + Task + Action + Result**

```
【案例結構】

S (Situation):情境
→ 客戶遇到什麼問題?

T (Task):任務
→ 客戶希望達成什麼目標?

A (Action):行動
→ 律師做了什麼?採取什麼策略?

R (Result):結果
→ 最後結果如何?客戶獲得什麼?

+ Testimonial:客戶見證
→ 客戶怎麼評價這次服務?
```

**六、隱私與真實性的平衡**

(一)法律與道德限制

**重要**:展示案例時要注意:

```
【可以做的】
✓ 描述案件類型(車禍、離婚、房屋糾紛)
✓ 說明處理方式與結果
✓ 使用化名或匿名(張先生、王小姐)
✓ 獲得客戶同意後,使用真名見證

【不能做的】
✗ 揭露客戶個資(全名、詳細地址、身分證字號)
✗ 洩漏案件機密內容
✗ 未經同意使用客戶照片
✗ 誇大不實(如:宣稱「100% 勝訴」)
```

**建議做法**:
1. 案例使用化名:「張先生」、「王小姐」
2. 地點模糊化:「台北」、「新北」(不寫詳細地址)
3. 重點在「類型」和「結果」,不在「八卦」
4. 客戶見證一定要徵求同意

---

(二)如何讓匿名案例看起來真實?

**技巧 1:細節描述**

```
❌ 太簡略(不真實)
「協助客戶處理車禍案件,獲得賠償。」

✅ 有細節(真實感)
「張先生騎機車被轎車擦撞,右腿骨折住院 2 個月,
 我們協助計算醫療費 18 萬、工作損失 24 萬、
 看護費 12 萬,最終獲賠 120 萬元。」
```

**技巧 2:時間與數字**

```
有具體時間:「3 個月內完成理賠」
有具體數字:「從 30 萬爭取到 120 萬」
有具體比例:「勝訴率 95%」
```

**技巧 3:客戶心聲**

```
真實的見證不會太完美:

❌ 假見證(太完美)
「陳律師是世界上最棒的律師!」

✅ 真實見證(有細節)
「陳律師很專業,最重要的是他用我聽得懂的方式
 解釋法律,讓我不會那麼緊張。」
```

### 7-4 線上諮詢表單:降低聯絡門檻

**七、為什麼需要線上諮詢表單?**

(一)傳統方式 vs 線上表單

**傳統方式(打電話)**:

```
客戶的內心戲:
「要打電話...好緊張...」
「不知道要說什麼...」
「會不會講錯什麼?」
「萬一律師很兇怎麼辦?」
「還是算了...」

結果:70% 的潛在客戶放棄聯絡
```

**線上表單方式**:

```
客戶的內心戲:
「有線上表單,先填填看」
「可以慢慢想要怎麼寫」
「不用講話,比較不緊張」
「送出就好,簡單!」

結果:轉換率提升 3-5 倍
```

**線上表單的優勢**:
- 降低心理門檻(不用即時對話)
- 客戶可以仔細思考(不會漏掉重點)
- 律師可以先評估案件(決定是否接案)
- 留下紀錄(不會遺漏資訊)

---

(二)好的諮詢表單設計

**設計原則**:

```
1. 欄位不要太多(不超過 8 個)
   → 太多會讓人放棄填寫

2. 標示哪些必填(用紅色星號 *)
   → 讓人知道至少要填什麼

3. 提供範例或提示
   → 降低填寫難度

4. 即時驗證格式
   → 避免送出後才發現錯誤

5. 明確說明「下一步是什麼」
   → 讓客戶知道送出後會發生什麼
```

---

**表單欄位設計**:

```
【基本資料】(必填)
姓名:__________ *
電話:__________ *
Email:_________ *

□ 房屋糾紛

□ 其他(請說明)

案件簡述:(文字框) *
請簡單描述您的問題(至少 50 字)

範例:
「我在 XX 路口被汽車撞到,對方闖紅燈,
 我腿部骨折住院一個月,保險公司只願意賠 10 萬,
 但我醫療費就花了 15 萬。請問我該怎麼辦?」

【期望協助】(選填)

**重點**:

## 重要名詞

◆ **信任指標 (Trust Indicators)**:網站上用來建立信任感的元素,如執業年資、勝訴率、客戶見證等。

◆ **STAR 法則**:案例展示的結構化方法,包含 Situation(情境)、Task(任務)、Action(行動)、Result(結果)。

◆ **社會證明 (Social Proof)**:利用他人的經驗和評價來影響決策的心理現象,客戶見證就是一種社會證明。

◆ **行動呼籲 (Call To Action, CTA)**:促使使用者採取特定行動的設計元素,如「免費諮詢」按鈕。

◆ **資訊不對稱 (Information Asymmetry)**:買賣雙方掌握的資訊不平等,專業服務業尤其明顯,客戶不懂法律/會計/醫療。

◆ **轉換率 (Conversion Rate)**:訪客完成特定目標的比例,如填寫諮詢表單的訪客比例。

◆ **RWD (Responsive Web Design)**:響應式網頁設計,網站能自動適應不同裝置(電腦、平板、手機)。
