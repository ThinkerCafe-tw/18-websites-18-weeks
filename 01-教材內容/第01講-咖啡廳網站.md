## 主文

### 1-1 AI 時代網站開發的新思維

在傳統的網站開發流程中，你需要學習 HTML、CSS、JavaScript 等程式語言，花費數週甚至數月才能完成一個簡單的網站。但在 AI 時代，這個流程被徹底改變了。

**一、從「學工具」到「解決問題」的思維轉變**

（一）傳統開發的困境

傳統網站開發要求開發者：

1. 精通多種程式語言
2. 理解複雜的框架和函式庫
3. 手動處理各種技術細節
4. 不斷除錯和測試

這導致：
- 學習曲線陡峭，入門門檻高
- 開發時間長，成本高昂
- 非技術背景者難以參與

（二）AI 協作開發的革命

AI 協作開發改變了遊戲規則：

1. **自然語言即程式語言**：用日常對話描述需求，AI 自動生成程式碼
2. **快速迭代**：30 分鐘內從零到上線
3. **降低門檻**：零程式基礎也能開發專業網站
4. **專注價值**：把時間花在理解需求和解決問題，而非語法細節

### 1-2 30 分鐘完整開發流程實戰

**二、咖啡廳網站的商業需求分析**

（一）目標客群與核心需求

一家咖啡廳需要網站的原因：

1. 展示品牌形象和店內環境
2. 提供菜單資訊，減少現場點餐時間
3. 讓客人能輕鬆找到聯絡方式和地址
4. 傳達咖啡廳的理念和故事

（二）網站必備的四個頁面

1. **首頁**：視覺衝擊 + 品牌印象
   - 大圖橫幅展示店內氛圍
   - 簡短的品牌標語
   - 清楚的導航選單
   
2. **菜單頁**：產品展示 + 價格資訊
   - 咖啡、輕食、甜點分類
   - 清楚的價格標示
   - 吸引人的產品照片

3. **關於我們**：品牌故事 + 理念傳達
   - 創辦故事
   - 咖啡豆來源
   - 經營理念

4. **聯絡我們**：聯絡資訊 + 地圖
   - 地址、電話、營業時間
   - Google Maps 嵌入
   - 社群媒體連結

**三、尋找靈感的技巧**

（一）關鍵字搜尋策略

在 Google 搜尋時，善用關鍵字組合：

1. "咖啡廳網站設計"
2. "cafe website inspiration"
3. "coffee shop web design"

配合搜尋技巧：
- 使用 Google Images 快速瀏覽
- Pinterest 尋找設計靈感
- Dribbble、Behance 觀摩專業作品

（二）截圖的藝術：抓住設計重點

好的截圖應該包含：

1. **整體版面配置**：導航列位置、內容區塊安排
2. **色彩風格**：主色調、配色方案
3. **視覺元素**：圖片風格、字體選擇
4. **特色功能**：互動效果、特殊設計

截圖技巧：
- 截取完整頁面，包含頂部和底部
- 標註特別喜歡的部分
- 準備 3-5 張參考圖片

### 1-3 與 AI 的第一次對話

**四、如何用自然語言描述需求**

（一）有效溝通的三個層次

1. **告訴 AI 你是誰**
   - "我是一家咖啡廳的老闆"
   - "我需要一個展示品牌的網站"

2. **描述具體需求**
   - "我需要四個頁面：首頁、菜單、關於、聯絡"
   - "首頁要有大圖橫幅，展示店內溫馨的環境"

3. **提供參考資料**
   - 上傳截圖
   - "我喜歡這個網站的配色和版面"

（二）實戰對話範例

```
你：我想做一個咖啡廳網站，需要四個頁面。首頁要有大圖橫幅和品牌介紹，
    菜單頁要分類展示產品和價格，關於頁面講我們的故事，聯絡頁面有地圖。
    整體風格要溫暖、簡約，像這些參考網站的感覺。[上傳截圖]

Claude：好的，我來幫你建立這個咖啡廳網站。我會使用 React 和 Tailwind CSS 
        來製作一個響應式網站...
```

（三）迭代優化的技巧

第一版出來後，可以這樣調整：

1. "首頁的字體可以大一點嗎？"
2. "菜單的產品圖片想要圓角設計"
3. "導航列改成深咖啡色背景"

### 1-4 部署上線：讓全世界看到你的作品

**五、GitHub + Vercel 的完整流程**

（一）為什麼選擇 Vercel？

Vercel 的優勢：

1. **完全免費**：個人網站零成本
2. **自動部署**：推送程式碼就自動上線
3. **全球 CDN**：網站速度快
4. **HTTPS 加密**：安全性有保障

（二）三個步驟完成部署

1. **建立 GitHub 儲存庫**
   - 註冊 GitHub 帳號
   - 新增 repository
   - 上傳網站程式碼

2. **連接 Vercel**
   - 註冊 Vercel 帳號
   - 連接 GitHub
   - 選擇要部署的專案

3. **取得專屬網址**
   - Vercel 自動產生 `your-project.vercel.app`
   - 可自訂網域名稱（需購買網域）

（三）手機測試的重要性

部署完成後，務必測試：

1. 手機瀏覽是否正常
2. 平板尺寸是否跑版
3. 不同瀏覽器的相容性
4. 載入速度是否流暢
