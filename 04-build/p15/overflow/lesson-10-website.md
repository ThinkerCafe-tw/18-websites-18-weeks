# 第 10 講 溢出素材 — 配套網站候選

> 本檔為第 10 講字數收斂時搬出的程式碼示意與技術細節，供配套網站產線使用。

---

> 來源：第10講 §5.2 AI 產出結果（首屏完整描述，正文已壓縮）

首頁第一屏包含教練名稱、定位與一句話主張，以及「預約免費諮詢」「先看方案與規則」兩個按鈕；往下依序是教練定位、學員案例、課程方案、諮詢流程、近期諮詢時段、規則與付款前說明、預約諮詢表單與頁尾。照片先以免費圖庫的示意圖代替，AI 也用註解標出了日後要替換的正式檔名，例如 `images/coach-profile.jpg`；證照、電話與 LINE 帳號則以「上線前填入」「上線前替換」的佔位文字標示，沒有編造資歷與評價。三張學員案例卡同樣只是示意內容，卡片上都標著「案例示意（上線前替換為真實案例）」，上線前需替換為取得學員同意的真實案例。

`main` 內依序是主視覺、教練定位（TrainerProfile 的名稱、地區、定位、服務時間）、學員案例（三張案例卡，各帶起點、目標、期間、方法、結果、限制）、課程方案（三個方案與各自的價格、內容、限制）、諮詢流程、近期諮詢時段（ScheduleSlot 的日期、時段、名額、狀態表格）、規則與付款前說明、預約諮詢表單八個 `section`。

> 來源：第10講 §5.2 AI 產出結果（HTML 骨架示意，正文已搬出）

整體結構可以用下面的骨架示意（省略號代表實際內容，不是可直接貼用的程式碼），這就是需求模組表落到程式碼後的樣子。其中「聯絡與後續」被 AI 併進詢問表單區塊與頁尾，「信任證據」則拆成教練定位與學員案例兩個區塊，都是常見而可接受的調整：

```html
<header>…教練名稱與導覽…</header>
<main>
  <section class="hero">…名稱、定位、主張、預約免費諮詢、先看方案與規則…</section>
  <section id="profile">…教練定位、服務時間、待填的證照欄位…</section>
  <section id="cases">…三張學員案例卡與佔位照片…</section>
  <section id="plans">…課程包、價格、有效期限…</section>
  <section id="process">…五步諮詢流程…</section>
  <section id="slots">…諮詢時段表格…</section>
  <section id="policy">…規則與付款前說明…</section>
  <section id="contact">…諮詢表單與聯絡方式…</section>
</main>
<footer>…服務地區、非醫療聲明、版權…</footer>
```

> 來源：第10講 §5.2 AI 產出結果（media query 節錄，正文已搬出）

手機版的關鍵是 CSS 裡的 media query。AI 產出的規則節錄如下：

```css
/* 平板：案例與方案卡片改為兩欄 */
@media (max-width: 1024px) {
  .cards, .plans { grid-template-columns: repeat(2, 1fr); }
}
/* 手機：全部單欄，按鈕與表單欄位維持易點 */
@media (max-width: 640px) {
  .cards, .plans { grid-template-columns: 1fr; }
}
```

把產出結果對回第 2 節的業態需求：學員案例保留起點、期間、方法與「個人結果不代表保證成果」的限制，回應「成功案例的多樣性」與被評判的恐懼；課程包的有效期限、退費規則與耗課紀錄，回應「金錢和成效脫節」的恐懼與買課之後的失控感；諮詢流程寫明「教練主要在聽」「不會當場推銷」，回應初次諮詢的「傾聽比例」；補充品揭露與非醫療聲明，則是 3.5 信任底線的落地。每個區塊都應該能對回一個需求；對不回去的區塊，就是可以刪掉的裝飾。

> 來源：第10講 §6.4 延伸學習（延伸書單與網站，正文已搬出）

**書籍**

麥可．博伊爾（Michael Boyle）著（二○一六）。*New Functional Training for Sports*（第二版）。Human Kinetics。── 功能性訓練方法的重要參考，有助於教練建立紮實的動作評估能力。

羅伯特．席亞爾迪尼（Robert Cialdini）著；閻紀宇譯（二○二三）。《影響力》（*Influence*，全新擴充版）。方智。── 本講心理學錨點與信任底線核心理論的完整出處。

**網站**

> 來源：第10講 §6.1 重要名詞（心理學四詞，與 §3.4 心理學錨點 / §3.5 信任底線正文重複，故由術語表搬出；正文已完整說明）

**信任三支柱**（Ability, Benevolence, Integrity）：梅爾等人（1995）的信任模型 — 信任來自對能力、善意與正直的同時評估。教練品牌就是在三個維度上提供可信訊號。

**社會證明**（Social Proof）：席亞爾迪尼（2006）的影響力原則 — 人在不確定時，以「和自己類似的人的選擇」作為行動依據。見證應優先展示起點的相似度。

**承諾一致性**（Commitment and Consistency）：人傾向讓後續行為與已做出的承諾一致。預付能提升出席意願；過度銷售大包課是其扭曲應用。

**倖存者偏差**（Survivorship Bias）：只展示成功案例造成對成功機率的高估。只呈現極端體態轉變是典型應用。
