# 第 09 講溢出素材 — 配套網站候選

> 來源：第9講 §5.2 AI 產出結果（首屏描述與完整產出清單，正文壓縮為定容版）

首頁第一屏包含店名、地區與「貓狗分區・全透明美容區」的定位、一句話主張，以及「預約美容」「看服務流程」兩個按鈕；往下依序是環境與信任證據、服務流程與預約規則、預約表單、規則說明、聯絡與後續。環境照片與地圖先以免費圖庫的示意圖代替，AI 也用註解標出了日後要替換的正式檔名，例如 `images/env-glass.jpg`。

這一版產出的內容完整清單：

- 一個 `index.html`，HTML 與 CSS（Cascading Style Sheets，階層式樣式表）寫在同一個檔案內。
- `header`：店名與導覽。
- `main` 內依序是主視覺、信任證據（環境照片、清潔做法、評論刊登原則、異常處理方式）、核心流程（五步服務流程、服務項目、可預約時段、預約規則）、預約表單（飼主與寵物資料、健康與行為備註、風險同意勾選）、規則說明、聯絡與後續六個 `section`。
- `footer`：店名、地區、營業時間與版權聲明。

> 來源：第9講 §5.2 AI 產出結果（media query 節錄）

手機版的關鍵是 CSS 裡的 media query。AI 產出的規則節錄如下：

```css
/* 平板：照片與服務卡片兩欄 */
@media (max-width: 1024px) {
  .grid, .cards { grid-template-columns: repeat(2, 1fr); }
}
/* 手機：全部單欄，表單欄位與按鈕維持易點 */
@media (max-width: 640px) {
  .grid, .cards { grid-template-columns: 1fr; }
}
```

用手機寬度打開同一個檔案，環境照片與服務卡片會改成單欄，按鈕也會放大成整排可點。

> 來源：第9講 §2.1 產業營運邏輯 — 健康哨兵轉化與高毛利產品（字數精煉時移出的高毛利結構細節）

護膚、深層 SPA、天然有機洗劑的毛利率，業界觀察在六到八成之間；當這類產品收入佔到整體營收三成，淨利結構會明顯改善。這就是「健康哨兵」轉化的商業終點：美容師在每次洗澡時做系統性健康觀察（皮膚乾濕、外耳道異味、趾間紅腫），把觀察轉成具體建議，例如「你家的狗皮膚這個月比較乾，建議試試深海礦物浴鹽」，接受率遠高於櫃檯式推銷。關鍵在於推薦的來源——來自觀察的建議，飼主感受到的是被照顧；來自話術的建議，飼主感受到的是被推銷。同一句產品名，放在這兩種語境下，轉化率與客訴率都完全不同。

> 來源：第9講 §2.1 產業營運邏輯 — 包月制與洗剪分流（字數精煉時移出的營運數字與檔案機制）

包月制的黏著真正鎖在連續健康檔案上：記錄皮膚狀況、指甲生長速度、耳垢的季節性變化三個月後，美容師接過寵物的第一分鐘就能說「你的狗上個月耳垢比較多，這個月有沒有比較好？」——這句話另一家店複製不了，因為它需要的是時間序列資料，不是技術。洗剪分流的人力結構同理：把工作拆成低薪助理負責的體力前段（洗澡、烘乾、清潔）與高薪美容師負責的技術後段（造型修剪、客戶溝通、健康評估），單一美容師全包時一天只能處理三隻，分流後同一位美容師的精緻修剪產出可拉到七到八件，產值接近翻倍，而新增成本只有一位助理的薪資。兩個機制疊在一起，才是老手算這門生意的方式。
