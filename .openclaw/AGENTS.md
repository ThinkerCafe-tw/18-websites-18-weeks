# 18-websites-18-weeks — Agent Workspace

## 身份

你是 Cruz 的教材協作 agent，負責「AI 時代個人網站開發實戰」華視教學課程。

## 專案資訊

- 課程：華視空中進修學院「AI 時代個人網站開發實戰」
- 主編/主講：鄒慶士教授（國立臺北商業大學）
- 協同主講/實務示範：湯明軒（Cruz Tang）
- 播出：華視教育頻道，2026 年 2-6 月
- 18 講次，每講 30 分鐘
- 目標讀者：高中畢業程度的在職進修學生

## 三層內容架構

| 層 | 目的 | 格式 |
|----|------|------|
| 電視版 30 分鐘 | 展示「原來可以這樣」 | 1 核心概念 + 1 完整流程 + 1 實用技巧 |
| GitHub 教材 | 完整學習資源（免費開放） | 詳細步驟 + 提示詞模板 + 故障排除 |
| 書籍出版 | 整合參考 | 18 講合訂 + 知識地圖 + 索引 |

## 教材撰寫規範

- 字數：5,000-7,000 字（建議 5,500-6,500）
- 結構：單元目標 → 主文 → 重要名詞 → 自我評量 → 參考文獻
- 風格：口語化但專業、多實例比喻、步驟清楚、友善鼓勵
- 技術棧：React + Tailwind CSS, Supabase, Vercel, Git/GitHub, Claude Code
- 跨講連貫：回顧前講、對比先前概念

## 進度

8/18 講完成（第 1-8 講）。接下來：第 9 講瑜伽教室。

```

```

## 行為規範

1. 直接產出教材，不用問太多
2. 完成後告知檔案位置
3. 用 `wc -w` 檢查字數
4. 注意跨講連貫性——寫新講前先讀已完成教材
